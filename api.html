API Documentation

Programmatic access to Universal Calculator Suite's 120+ calculator tools

Introduction

The Universal Calculator Suite API allows developers to integrate our calculator tools into their applications. Our API provides programmatic access to 120+ calculators across multiple categories.

Quick Start

POST /api/v1/calculate 
curl -X POST https://api.universalcalculatorsuite.com/v1/calculate \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "calculator": "mortgage",
    "params": {
      "loanAmount": 300000,
      "interestRate": 4.5,
      "loanTerm": 30
    }
  }'

Authentication

All API requests require an API key. You can obtain one by:

  1. Creating an account on our developer portal
  2. Generating an API key in your dashboard
  3. Including the key in the Authorization header
Authorization: Bearer YOUR_API_KEY_HERE

Available Calculators

  • roi - Return on investment

Health Calculators

  • bmi - Body Mass Index
  • calories - Daily calorie needs
  • body-fat - Body fat percentage
  • ideal-weight - Healthy weight range
  • water-intake - Daily water needs

Math Calculators

  • percentage - Percentage calculations
  • fraction - Fraction operations
  • algebra - Algebraic equations
  • geometry - Area and volume
  • statistics - Mean, median, mode

API Endpoints

GET /api/v1/calculators

List all available calculators

POST /api/v1/calculate

Perform a single calculation

POST /api/v1/calculate/batch

Perform multiple calculations in one request

GET /api/v1/calculators/{calculator_id}/schema

Get parameter schema for a specific calculator

Example Request: Mortgage Calculator

{
  "calculator": "mortgage",
  "params": {
    "loanAmount": 300000,
    "downPayment": 20,
    "interestRate": 4.5,
    "loanTerm": 30,
    "propertyTax": 3600,
    "insurance": 1200
  }
}

Error Codes

SDK Libraries

We provide official SDKs for popular programming languages:

JavaScript/Node.js

npm install @universal-calculator/sdk
const { CalculatorClient } = require('@universal-calculator/sdk');

const client = new CalculatorClient('YOUR_API_KEY');

const result = await client.calculate('mortgage', {
  loanAmount: 300000,
  interestRate: 4.5,
  loanTerm: 30
});

Python

pip install universal-calculator
from universal_calculator import CalculatorClient

client = CalculatorClient(api_key='YOUR_API_KEY')
result = client.calculate(
    calculator='bmi',
    params={'height': 170, 'weight': 70}
)
